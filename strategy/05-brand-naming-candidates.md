## 브랜드 포지셔닝 요약

- **타겟**: 한국계/아시아계 이민자 식품 창업가 → 캐나다 전체 SMB → 미국 확장
- **퍼스널리티**: Approachable, Reliable, Practical, Inclusive, Modern
- **톤**: 쉽고, 진지하지 않고 간단하지만, 확실하고 군더더기 없는 식품/건기식 시작 자문 플랫폼

## 5가지 브랜드명 후보

### 1. FreshPath (추천 1위)

| 항목 | 내용 |
|------|------|
| **의미** | Fresh (신선한/새로운) + Path (길/경로) — 식품업계 진입을 위한 새롭고 깨끗한 길 |
| **전략적 이유** | 식품 산업 연관성(Fresh)과 가이던스(Path) 결합. 규제 "clearance"를 "path"로 재해석 — 더 긍정적이고 forward-looking. 컴플라이언스 → 비즈니스 자문 → 시장 리서치로 확장 가능 |
| **발음** | EN: /frɛʃ pæθ/ · KR: "프레시패스" |
| **태그라인** | "Your fresh path to food success" / "식품 창업의 새로운 길" |
| **도메인** | freshpath.com ⚠️ 점검 필요 · freshpath.ca ✅ · freshpath.io ✅ |
| **장점** | 기억하기 쉬움, 긍정적 어감, 확장성 높음, 식품과 자연스러운 연관 |
| **단점** | 규제/컴플라이언스 연관성이 직접적이지 않음 |
| **비주얼** | 상승하는 곡선/화살표, 신선한 그린 + 모던 블루 조합 |

---

### 2. ClearCourse (추천 2위)

| 항목 | 내용 |
|------|------|
| **의미** | Clear (명확한/통과) + Course (과정/방향) — 명확한 진로와 과정을 제시 |
| **전략적 이유** | "Clear" → 규제 clearance + 명확성 모두 표현. "Course" → 교육적 가치와 방향성. 기존 "ClearBite"와 브랜드 연결성 유지 가능 |
| **발음** | EN: /klɪr kɔrs/ · KR: "클리어코스" |
| **태그라인** | "Chart your clear course to market" / "시장 진출의 명확한 항로" |
| **도메인** | clearcourse.com ⚠️ 교육 업체와 혼동 가능 · clearcourse.ca ✅ · clearcourse.io ✅ |
| **장점** | 기존 브랜드와 연결성, 전문성+친근함 균형, 확장성 좋음 |
| **단점** | 교육 플랫폼과 혼동 가능 |
| **비주얼** | 나침반, 항해 테마, 클린한 네이비-화이트 조합 |

---

### 3. FlowFree (추천 3위)

| 항목 | 내용 |
|------|------|
| **의미** | Flow (흐름) + Free (자유로운/방해받지 않는) — 규제 장벽 없이 자유롭게 흐르는 비즈니스 |
| **전략적 이유** | 규제의 복잡함을 "flow" 개념으로 단순화. "Free" → 자유로움과 비용 효율성 모두 암시. YC 스타일의 트렌디한 느낌 |
| **발음** | EN: /floʊ fri/ · KR: "플로우프리" |
| **태그라인** | "Flow free to success" / "자유롭게 흘러가는 성공" |
| **도메인** | flowfree.com ⚠️ 게임 앱 사용 중 · flowfree.ca ✅ · flowfree.io ✅ |
| **장점** | 가장 모던하고 트렌디, 확장성 최고, 긍정적 어감 |
| **단점** | 규제 전문성이 덜 명확, 너무 추상적일 수 있음 |
| **비주얼** | 유체 흐름, 곡선, 자유로운 그라데이션, 블루-퍼플 조합 |

### 4. RuleBridge

| 항목 | 내용 |
|------|------|
| **의미** | Rule (규칙/규제) + Bridge (다리) — 복잡한 규제와 창업가 사이를 연결하는 다리 |
| **전략적 이유** | 규제 전문성을 직접적으로 표현. "Bridge"로 접근성과 연결성 강조. B2B 신뢰성과 친근함의 균형 |
| **발음** | EN: /rul brɪdʒ/ · KR: "룰브릿지" |
| **태그라인** | "Bridge the gap to compliance" / "규제와 성공을 잇는 다리" |
| **도메인** | rulebridge.com ⚠️ 점검 필요 · rulebridge.ca ✅ · rulebridge.io ✅ |
| **장점** | 명확한 포지셔닝, 전문성 표현, 기억하기 쉬움 |
| **단점** | 다소 기능적 느낌, 컴플라이언스 외 확장 시 제한 |
| **비주얼** | 모던한 다리 실루엣, 연결선, 신뢰감 블루-그레이 팔레트 |
